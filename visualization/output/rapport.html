📊 Partie II.A — MapReduce sur Netflix

Dataset : Netflix Movies & TV Shows

Kaggle 8,807 entrées 12 colonnes

Colonnes : show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description

Objectifs d'analyse

  • Distribution des contenus par genre et type
  • Tendances de production par pays et année
  • Durée moyenne des films par genre

⚡ Partie II.B — Twitter Streaming (PySpark)

Architecture du Pipeline

Twitter API v2 → Tweepy StreamingClient → Socket TCP (port 9999) → PySpark DStream → TextBlob + MLlib LogisticRegression → Visualisation

🤖 Algorithme ML — Logistic Regression

Pipeline Spark MLlib :

Tokenizer → StopWordsRemover → HashingTF (1000 features) → IDF (TF-IDF weighting) → LogisticRegression (C=100, maxIter=20)


Lancement streaming :

python spark/streaming_twitter.py --simulate

Avec vraies clés API :

export TWITTER_BEARER_TOKEN="..." && python spark/streaming_twitter.py
